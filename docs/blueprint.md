# **App Name**: Mariya Makeup Artistry

## Core Features:

- Contact Strip Display: Display phone, email, Instagram and a 'Book Now' CTA on the header.
- Portfolio Gallery: Showcase bridal and commercial makeup work through interactive image galleries.
- Automated Brand Tag Generation: Employ an AI tool to create descriptive and relevant brand tags for new portfolio entries to make brand indexing on blogs much easier.
- Stats Display: Highlight key stats like number of brides, years of experience, and brands collaborated with.
- Testimonial Carousel: Display client reviews with a carousel or card slider for an interactive user experience.
- Blog Integration: Display featured blogs with makeup tips, bridal trends, and skin prep guides.
- Contact Form Submission: Enable potential clients to reach out for bookings through the application.

## Style Guidelines:

- Primary color: Soft blush (#E0A6B3), evoking femininity and luxury.
- Background color: Light beige (#F5F5DC), providing a neutral, clean base.
- Accent color: Muted gold (#BDB76B), adding subtle high-end accents and highlights.
- Body font: 'PT Sans', a modern sans-serif with a little warmth. Headline Font: 'Playfair', a serif with an elegant, fashionable, high-end feel.
- Use elegant, minimalist icons, potentially with fine line weights, to represent navigation items and key features.
- Emphasize an image-focused layout with clean lines and generous white space to enhance the visual appeal and luxury feel.
- Implement smooth scrolling and subtle transitions to give a polished and professional feel to user interactions.
